# 📖 漫畫角色介紹編輯器

**Manga Character Introduction Editor**

一款純前端、零依賴的單頁應用，用來製作日式漫畫風格的「登場人物紹介」版面。  
所有操作均在瀏覽器本地完成，不需伺服器、不會上傳任何資料。

## ⚙️ 功能說明

| 選項 | 說明 |

| 選項 | 說明 |

### 👥 角色管理

#### 新增 / 刪除角色

- 點擊 **＋ 新增角色** 增加卡片
- 滑鼠移到卡片右上角，點擊紅色 **×** 刪除

#### 自動排列

1. 選擇欄數（1–4欄）
2. 調整排列間距
3. 點擊 **⊞ 自動排列** → 卡片自動依網格分布

#### 自由定位（XY 軸）

每個角色列表下方有 **X / Y 座標輸入框**：

```
X: 129   Y: 120
```

- 直接輸入數值精確定位
- 或用滑鼠拖曳卡片頂部的 **⠿ ⠿ ⠿** 把手自由移動
- 拖曳時 XY 數值會即時同步更新

#### 編輯文字

在畫布上直接點擊角色名稱或說明文字即可編輯：

- **名稱**：單行，`Enter` 確認
- **說明**：多行，`Enter` 換行，可輸入多行描述

---

### 🖼 圖片設定

每個角色支援兩種圖片來源方式：

#### 方式一：上傳本機圖片

1. 點擊畫布上的圖框（灰色區域）
2. 或點擊側邊欄角色列表中的 **📁** 按鈕
3. 選擇本機圖片檔案（JPG、PNG、WebP…）

#### 方式二：輸入外部 URL / 路徑

在側邊欄角色列表的 **URL** 欄位輸入：

```
https://example.com/chara.png
```

或相對路徑：

```
./images/character1.png
```

輸入後按 `Enter` 或點擊其他地方即套用。

> 💡 URL 優先於上傳圖片顯示。若兩者都有，會使用 URL。

#### 圖片調整

上傳或設定圖片後：

| 動作 | 效果 |
|------|------|
| **拖曳圖片** | 在圖框內移動圖片位置 |
| **滾輪** | 縮放圖片大小 |
| **雙擊圖片** | 移除圖片（還原為空白） |
| **🔄 按鈕**（hover 時出現）| 替換圖片 |

## 💾 匯出格式

### PNG 輸出

- 2× 解析度高清點陣圖
- 保留所有圖片與樣式
- 適合社群媒體發布、存檔

### JPG 輸出

- 92% 品質壓縮點陣圖
- 檔案比 PNG 更小
- 適合一般分享用途

### SVG 輸出（向量）

- 可無限縮放不失真
- 包含文字、圖形路徑、嵌入圖片
- 適合後製、印刷用途

### JSON 設定檔

儲存目前所有設定，包含：

- 版面尺寸、顏色、字體
- 所有角色的位置、名稱、說明
- 圖片 URL（若使用 URL 方式）
- 圖框設定

**載入方式**：點擊 **⬆ 載入 JSON** 選擇 `.json` 檔案，即可完整恢復編輯狀態。

> ⚠️ 以本機上傳的圖片（base64）也會一起儲存在 JSON 中，檔案可能較大。

---

### ⟨/⟩ HTML 片段輸出（Caveduck / AIRP 專用）

點擊 **HTML 片段** 按鈕，彈窗顯示可複製的 HTML 代碼。

**符合規格**：

- ✅ 僅輸出可嵌入的 HTML 片段（無 `<!DOCTYPE>`、`<html>`、`<style>`、`<script>`）
- ✅ 僅使用 `<div>`、`<span>`、`<img>`、`<br>` 等允許標籤
- ✅ 無 `class`、`id`、`data-*` 屬性
- ✅ 全部使用 inline `style`
- ✅ `font-size`、`padding`、`margin` 使用 `rem` 單位
- ✅ 尺寸與定位使用 `px` / `%`

**Caveduck 貼上方式**：

```html
<!-- 直接貼上這樣的片段 ↓ -->
<div style="position:relative;width:794px;...">
  ...
</div>
```

> ❌ 不要包在 Markdown code block 裡  
> ❌ 不要貼含有 `class="..."` 的舊版輸出

點擊頂部 **🖨 列印 / PDF** 按鈕，呼叫瀏覽器列印對話框：

## 🔁 建議工作流程

```
1. 選擇版面尺寸（A4 / 自定義）
        ↓
2. 設定標題文字與字體
        ↓
3. 設定欄數，點擊「自動排列」
        ↓
4. 為每個角色：
   - 輸入名稱與說明
   - 上傳圖片或填入 URL
   - 滾輪縮放、拖曳調整構圖
        ↓
5. 調整外框形狀與顏色
        ↓
6. 微調 XY 座標或拖曳至理想位置
        ↓
7. 選擇匯出格式：
   PNG/JPG → 圖片分享
   SVG     → 向量後製
   JSON    → 存檔備份
   HTML    → 嵌入 Caveduck / AIRP
```

---

## 📝 注意事項

- **本機圖片**在匯出 JSON 時會轉為 base64 一起儲存，若圖片較多 JSON 檔案會偏大
- **SVG 匯出**中的文字使用系統字體備援，在不同裝置上字型可能略有差異
- 畫布採 `overflow: hidden`，卡片拖曳至邊界外的部分在匯出時不會顯示
- Caveduck HTML 輸出中使用 `clip-path`，請確認目標平台的瀏覽器支援度
